AutoRef.js

How to use:

Just add the following code after ending the <body> tag -

What it does:

Just place 'ref' attribute in your anchor (<a>) tag to open the url in the destination as it's value

Example -
<!−−This will open the url in the same destination as it's value−−>
<a ref>https://www.google.com</a>

Output - https://www.google.com
